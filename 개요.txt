Back Office



판매자가 상품을 등록하고 등록된 상품에 카테고리/셀링포인트/쿠폰 적용가능

각각의 기능은 CRUD를 가짐



- 상품관리

상품을 등록, 수정 시 카테고리, 셀링포인트, 쿠폰을 적용할 수 있음

상품은 적어도 하나 이상의 가격이력을 가졌고, 상품 가격은 해당 기간에 맞는 가격을 가짐

상품을 조회할 때, 상품이름이나 카테고리로 검색할 수 있으며 각 상품의 판매상태, 가격에 따른 매출액 등을 조회 가능 



- 카테고리관리

Anchor 카테고리 방식으로 대/중/소 분류를 나누었고

각 카테고리를 클릭할 때마다 해당 카테고리와 하위 카테고리에 적용된 상품 리스트가 조회됨

그리고 해당 카테고리에 상품을 적용할 수 있고 카테고리의 정보를 변경할 수 있음



- 쿠폰관리

쿠폰 등록, 수정 시 상품, 카테고리, 멤버등급, 유효기간, 발급기간 등을 설정 가능



- 셀링포인트관리

상품에 셀링포인트를 적용할 때 우선순위에 따라 상품 이름 앞, 뒤에 하나씩 적용 가능

각각의 셀링포인트가 어떤 상품에 적용되었는지 알 수 있음, 또 추가 및 변경 시 적용할 상품을 등록/수정 가능



- 주문관리

특정 기간에 따라 주문조회가 가능하며, 각 주문의 주문금액, 할인금액, 결제금액, 결제방식 등 조회 가능



- 회원등급관리

각 등급의 산정기준을 변경 가능 단, 산정기준의 범위가 겹치지 않고 끊이지 않아야 함

Front Office



메인페이지에는 bo에서 등록한 상품이 보여지고 priceMgr을 통한 정보들을 볼 수 있음

로그인이 필요한 기능이 있고 마이페이지에서는 사용자의 주문내역 및 발급쿠폰을 조회 가능



- 헤더

홈, 로그인, BO페이지, 마이페이지, 장바구니로 페이지 이동

상품이름 및 카테고리 분류로 검색하는 기능



- 상품리스트

기본적으로 전체 상품이 조회되며, 한 번에 10개씩 볼 수 있음

헤더의 카테고리 검색으로 상품이 조회되기도 하며,



- 상품상세

상품 클릭 시 상품 상세 페이지로 넘어가며 해당 상품과 관련된 정보들과 priceMgr이 적용된 최대할인가 조회 가능

해당 상품의 쿠폰을 다운받을 수 있으며 쿠폰을 적용할 수도 있음

쿠폰을 적용할 때에는 해당 상품에 적용할 수 있는 쿠폰 중 가장 할인이 많이되는 쿠폰이 자동으로 선택되어 있음



- 장바구니

유저가 담아놓은 상품들을 조회 가능하며, 각 상품별로 쿠폰을 적용 가능

각 상품별 상품금액, 할인금액, 결제금액 등의 정보 조회 가능 

상품의 수량 변경 가능 변경시 쿠폰 내역 초기화

체크선택으로 상품을 삭제하거나 선택 구매 가능



- 주문/결제

하나의 상품을 바로 구매하거나, 장바구니에 있는 상품들을 구매했을 때 주문/결제 페이지로 넘어감

주문/결제 페이지에서는 총 주문금액, 할인금액, 결제금액과 주문자 정보를 볼 수 있음



- 주문완료

주문/결제가 정상적으로 완료되면 해당 주문번호를 띄어줌
